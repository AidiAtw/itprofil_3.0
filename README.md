# ITprofil 3.0

### Typy commitů

- **feat** – nová funkce
- **fix** – oprava chyby
- **style** – úprava vzhledu kódu (CSS, formátování)
- **refactor** – přepsání kódu bez změny chování
- **docs** – dokumentace
- **chore** – údržba a konfigurace

### Příklady commitů

```bash
git commit -m "feat(profil): přidat datovou strukturu"
git commit -m "feat(php): vytvořit aplikaci s JSON"
git commit -m "style(css): přidat designu"
git commit -m "fix(php): opravit bezpečnost"
git commit -m "docs: aktualizovat README"
```
